SAVANNAH WILDLIFE ANALYTICS · DASHBOARD

A data-driven view of wildlife movement, sightings, and migration patterns.

This prototype simulates a fictional savannah ecosystem where lions, elephants, zebras, and giraffes are tracked across several watering holes and migration routes. The dashboard combines sightings, movement intensity, and seasonal migration flows to illustrate how analytics can support conservation decisions.

Tech stack: HTML, CSS, Bootstrap, Chart.js (simulated data)

Use cases: monitoring species activity, identifying high-pressure zones, and understanding how seasonality affects movement across the savannah.
